About
=====
F5 version 2. Easier to use.

Install
=======
    # r2 hasn't been published to npm yet,
    # but you can install from github
    $ npm Install unclebill/r2 -g

How to use
==========

    $ cd your-folder
    $ f5r2
    # r2 will run on port 3000 by default
